🔧 MCP Tools
use_free_llm
Call any free LLM API. Automatically routes to the correct provider based on model ID, with intelligent fallback on failure and LRU response caching.
model * messages * temperature max_tokens top_p provider fallback stream
list_available_free_models
List all free LLM models and their providers. Filter by provider ID or show only providers with configured API keys.
provider available_only
code_mode
Execute JavaScript code in a sandboxed QuickJS environment. Input data is available via the DATA variable. Use print() to output results. Supports timeout control and tracks compression ratios.
code * data command timeout_ms

🚀 Quick Start
# Install dependencies
npm install

# Configure API keys
cp .env.example .env
# Edit .env with your keys

# Run the MCP server
npm run dev

# Build for production
npm run build && npm start

# Run tests
npm test
// Claude Desktop config (claude_desktop_config.json)
{
  "mcpServers": {
    "free-llm-apis": {
      "command": "node",
      "args": ["/path/to/mcp-server/dist/server.js"],
      "env": {
        "GROQ_API_KEY": "your-key-here",
        "GEMINI_API_KEY": "your-key-here"
      }
    }
  }
}
